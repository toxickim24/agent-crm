## Animation Guidelines

### Entry Animations
- Fade in: 200ms ease-out
- Slide up: 300ms ease-out
- Scale in: 250ms ease-out

### Data Updates
- Number count-up: 500ms
- Chart transitions: 400ms ease-in-out
- Progress bars: 600ms ease-out

### Hover Effects
- Scale up: 150ms
- Shadow grow: 200ms
- Color change: 200ms
